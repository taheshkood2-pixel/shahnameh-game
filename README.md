# 🏛 شاهنامه: ظهور شاهان (Shahnameh: Rise of Kings)

بازی موبایلی **Idle RPG + Gacha + Kingdom Builder** بر اساس اسطوره‌های شاهنامه فردوسی و تاریخ ایران.

## 🎮 امکانات

- ✅ **۲۶ قهرمان اسطوره‌ای** با ۵ کلاس (جنگجو، تانک، تیرانداز، جادوگر، پشتیبان)
- ✅ **۲۴ دشمن و باس** — از ضحاک تا اژدهاک
- ✅ **۷ فصل داستانی کامل** با ۳۵+ مرحله و کات‌سین بر پایه‌ی شاهنامه
- ✅ **حالت بی‌نهایت** — بازی هرگز تموم نمی‌شه
- ✅ **سیستم اینونتوری کامل** با ۲۳ آیتم:
  - ⚔ سلاح‌ها (شمشیر، گرز گاوسر، کمان آرش، نیزه)
  - 👑 کلاه‌خود و تاج (تاج کیانی افسانه‌ای)
  - 🛡 زره (ببر بیان مخصوص رستم)
  - 👢 چکمه (نعل رخش)
  - 💍 انگشتر و تعویذ (مهره سیمرغ)
- ✅ **تجهیزات مخصوص** — هر آیتم روی قهرمان مربوطه‌اش ۲۰٪ بونس اضافه می‌ده
- ✅ **گاچا (Gacha)** با نرخ SSR/SR/R و انیمیشن احضار
- ✅ **پایتخت پارسه** — ۶ بنا با ارتقای بی‌نهایت
- ✅ **درآمد آفلاین** — طلا حتی وقتی بازی بسته‌ست
- ✅ **دوبله فارسی حماسی**
- ✅ **دوزبانه فا/EN** با یک کلیک
- ✅ **PWA + آفلاین** با Service Worker
- ✅ **RTL/Mobile-first** — بهینه برای گوشی

## 🚀 نصب سریع

### راه ۱: بازی مستقیم در مرورگر
`index.html` رو باز کن. تمام.

### راه ۲: نصب روی گوشی (بدون APK)
1. فایل‌ها رو در GitHub Pages / Netlify میزبانی کن
2. با Chrome موبایل باز کن → منو → **Add to Home Screen**

### راه ۳: APK خودکار با گیت‌هاب
راهنمای کامل: [**GITHUB_SETUP.md**](./GITHUB_SETUP.md)
یا مستقیماً:
```bash
bash push-to-github.sh YOUR_USERNAME
```

## 📁 ساختار

```
shahnameh_game/
├── index.html          ← UI بازی
├── game.js             ← موتور بازی (۷۰۰+ خط)
├── data.js             ← دیتای بازی (قهرمانان، دشمنان، آیتم‌ها، داستان)
├── manifest.json       ← تنظیمات PWA
├── sw.js               ← Service Worker
├── push-to-github.sh   ← اسکریپت انتشار
├── GITHUB_SETUP.md     ← راهنمای گیت‌هاب
├── BUILD_APK.md        ← ۴ روش ساخت APK
├── .github/workflows/  ← GitHub Actions (خودکار APK می‌سازه)
└── assets/
    ├── heroes/         ← ۲۶ پرتره قهرمان
    ├── enemies/        ← ۲۴ باس
    ├── items/          ← ۲۳ آیتم
    ├── bg/             ← پس‌زمینه
    ├── ui/             ← لوگو
    ├── story/          ← پس‌زمینه‌ی کات‌سین‌ها
    └── audio/          ← دوبله فارسی
```
